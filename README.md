# Customer-Behavior-Assessment
Customer Shopping Behavior Analysis - Python, SQL &amp; Power BI

This project presents a complete end-to-end data analysis pipeline that explores customer shopping patterns using Python (EDA), SQL (analytics), and Power BI (dashboard visualization). It uncovers key insights about customer demographics, spending behavior, product performance, loyalty segments, and the impact of discounts and subscriptions on overall revenue.
